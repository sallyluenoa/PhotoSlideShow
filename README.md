## Tools

## Project Framework

```
org.fog_rock.photo_slideshow
  ├ app: アプリケーションUIに関するモジュール
  │  ├ main: スライドメイン画面
  │  ├ menu: メニュー画面
  │  ├ module: UIに関連するライブラリモジュール  
  │  ├ select: アルバム選択画面
  │  └ splash: 起動画面
  └ core: ライブラリやデータ等のサポートモジュール
     ├ file: ファイル関連の処理
     ├ entity: データクラス
     ├ viper: VIPERモジュールインターフェース
     └ webapi: Web API 関連
```
