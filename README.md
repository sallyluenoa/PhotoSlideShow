## Project Framework

```
org.fog_rock
  ├ app: アプリケーションUIに関するモジュール
  │  ├ main: スライドメイン画面
  │  ├ menu: メニュー画面
  │  ├ select: アルバム選択画面
  │  └ splash: 起動画面
  └ core: ライブラリやデータ等のサポートモジュール
     ├ file: ファイル関連の処理
     ├ entity: データクラス
     ├ googleapi: Google API 関連
     └ viper: VIPERモジュールインターフェース
```
